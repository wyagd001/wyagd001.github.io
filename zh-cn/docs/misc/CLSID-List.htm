CLSID List (Windows Class Identifiers)

操作系统中某些特殊的文件夹使用唯一的字符串进行标识. 这些字符串的其中一些可以用在 FileSelectFile 和 FileSelectFolder. 例如:

FileSelectFile, OutputVar,, ::{645FF040-5081-101B-9F08-00AA002F954E}  ; 在回收站中选择文件.
FileSelectFolder, OutputVar, ::{20D04FE0-3AEA-1069-A2D8-08002B30309D}  ; 在此电脑中(以前是我的电脑或计算机) 选择文件夹.

要通过 Run 打开 CLSID, 只需将 CLSID 指定为第一个参数即可. 下表中的大多数 CLSID 都可以通过 Run shell:CLSID 来打开. 有些可以不使用 shell: 前缀来打开. 例如:

Run shell:::{D20EA4E1-3957-11D2-A40B-0C5020524153}  ; 管理工具.
Run ::{21EC2020-3AEA-1069-A2DD-08002B30309D}  ; 所有控制面板项.
Run ::{20D04FE0-3AEA-1069-A2D8-08002B30309D}  ; 此电脑(以前是我的电脑或计算机).
Run ::{645FF040-5081-101B-9F08-00AA002F954E}  ; 回收站.
Run ::{450D8FBA-AD25-11D0-98A8-0800361B1103}\My Folder  ; 打开我的文档中存在的文件夹.
Run %A_MyDocuments%\My Folder  ; 在大多数系统上与上述相同.

CLSID 的可用性和功能因当前操作系统, 安装的应用程序和使用的命令而异. 可用性一栏表示 CLSID 可用于的操作系统版本范围(XP, Vista, 7, 8, 10, 11). 其他 CLSID 可在 HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID 注册表中找到.
