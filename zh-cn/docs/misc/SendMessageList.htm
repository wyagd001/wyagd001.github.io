Windows 消息

Windows 消息用于在操作系统和应用程序之间, 以及应用程序的不同部分之间进行通信.

Windows 消息只是一个指定特定事件的数字代码. 例如, 如果用户按下鼠标左键, 窗口会收到一条消息, 消息代码如下: WM_LBUTTONDOWN(0x0201).

有些消息具有与之关联的数据. 例如, WM_LBUTTONDOWN message 消息包含鼠标光标的 X 和 Y 坐标.

消息号的范围如下:

  • 0 到 0x03FF: 保留给系统使用的消息.
  • 0x0400(WM_USER) 到 0x7FFF: 由私有窗口类使用的整数消息.
  • 0x8000(WM_APP) 到 0xBFFF: 供应用程序使用的消息. 此范围内的消息不会与系统消息冲突.
  • 0xC000 到 0xFFFF: 供应用程序使用的字符串消息. 此范围内的消息是在运行时通过系统的 RegisterWindowMessage 函数定义的.
  • 大于 0xFFFF: 表示系统保留.

有关详情, 请参阅关于消息和消息队列(Microsoft Docs).

在 AutoHotkey 中, 消息号可用于 PostMessage, SendMessage 和 OnMessage 的 MsgNumber 参数. 下面是一些值的列表. 要了解如何使用特定消息(例如 WM_VSCROLL) 的更多信息, 请在 Microsoft Docs 或您选择的搜索引擎中查找它. 另外, 请查看消息教程.
