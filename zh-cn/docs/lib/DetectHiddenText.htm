DetectHiddenText

设置在探测窗口时窗口中隐藏的文本是否 "可见". 此设置影响命令, 内置函数和控制流语句, 如 WinExist() 和 WinActivate.

DetectHiddenText, OnOff

参数

OnOff

指定以下单词之一:

On: 检测隐藏文本.

Off: 不检测隐藏文本.

[v1.1.30+]: 可以分别使用十进制值 1 和 0 代替 On 和 Off.

备注

如果不使用 DetectHiddenText, 默认设置为 On.

"隐藏文本" 是与窗口中那些不可见控件有关的一个术语. 它们的文本因此被认为是 "隐藏的". 当您想检测多窗格窗口或多选项卡式对话框中不同窗格的差异时, 可能需要关闭 DetectHiddenText. 使用 Window Spy 可以判断当前活动窗口中有哪些隐藏文本. 此设置会影响所有接受 WinText 参数的命令, 内置函数和控制流语句, 包括 WinActivate, WinActive(), IfWinActive, WinWait, WinExist() 和 IfWinExist.

内置变量 A_DetectHiddenText 包含了当前的设置(On 或 Off).

每个新运行的线程(例如热键, 自定义菜单项或定时子程序) 都会以此命令的默认设置开始. 通过在脚本的自动执行段(脚本的顶部) 使用此命令可以改变这个默认设置.
