StrLen()

获取字符串中包含的字符数量.

Length := StrLen(String)

参数

String
要统计的字符串.

返回值

函数返回字符串的长度.

备注

如果 String 是由 ClipboardAll 之前所赋值的变量, 那么 StrLen() 将报告其总大小.

相关

StringLen, InStr(), SubStr(), Trim(), StringLower, StringUpper, StrPut(), StrGet(), StrReplace(), StrSplit()

示例

获取和报告字符串中包含的字符数量.

StrValue := "The quick brown fox jumps over the lazy dog"
MsgBox % "The length of the string is " StrLen(StrValue) ; 结果为: 43
