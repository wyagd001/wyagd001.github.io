#EscapeChar

改变脚本的转义字符(例如, 重音符与反斜杠).

过时的: 不推荐在新脚本中使用此指令. 请使用默认的转义字符(重音/反单引号) 代替.

#EscapeChar NewChar

参数

NewChar

指定单个字符.

备注

转义符用来表明在其后紧跟着的字符不能像平常那样进行解释.

如果脚本中未使用此指令, 那么它会表现得就像设置为重音符(`), 在大多数英语键盘的左上角. 使用这个字符而不是反斜杠, 以避免在文件路径中需要两个反斜杠.

有关序列及其结果的完整列表, 请参阅转义序列(例如 `n 将产生一个换行符).

和其他指令一样, #EscapeChar 不能有条件地执行.

相关

还有下面这些极少使用的指令; 它们的用法如这些例子所示:

#DerefChar #  ; 把默认的引用符号 % 改为此符号.
#Delimiter /  ; 把默认的分隔符号逗号改为此符号.

过时的: 不推荐在新脚本中使用这些指令. 请使用默认的解除引用字符(百分号) 和默认的分隔符(逗号) 代替.

示例

把默认的转义字符(`) 改为反斜杠.

#EscapeChar \
