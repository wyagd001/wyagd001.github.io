StringTrimLeft / StringTrimRight

从字符串的左边或右边移除一些字符.

过时的: 不推荐在新脚本中使用这些命令. 请使用 SubStr 函数代替.

StringTrimLeft, OutputVar, InputVar, Count
StringTrimRight, OutputVar, InputVar, Count

参数

OutputVar

用来存储 InputVar 缩短后字符串的输出变量的名称.

InputVar

需要被读取内容的输入变量的名称. 不要把名称括在百分号中, 除非您希望使用变量的内容作为被解析的变量名.

Count

需移除的字符数, 可以为表达式. 如果 Count 小于或等于零, 那么 OutputVar 会被置为与 InputVar 相同. 如果 Count 超过了 InputVar 的长度, 那么 OutputVar 会被置空.

备注

在这个及其他所有命令中, OutputVar 和 InputVar 中可以使用相同的变量.

相关

SubStr(), StringMid, StringLeft, StringRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

示例

从左侧删除 5 个字符, 并将字符串 "is a test." 存储在 OutputVar.

String := "This is a test."
StringTrimLeft, OutputVar, String, 5

从右侧删除 6 个字符, 并将字符串 "This is a" 存储在 OutputVar.

String := "This is a test."
StringTrimRight, OutputVar, String, 6
