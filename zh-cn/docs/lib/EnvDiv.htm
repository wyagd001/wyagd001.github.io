EnvDiv

设置变量的值为它自身除以给定的值. 等同于: Var /= Value

过时的: 不推荐在新脚本中使用此命令. 而是使用 Var := Var / Value 或 Var /= Value.

EnvDiv, Var, Value

参数

Var

要运算的变量名.

Value

任何整数, 浮点数或表达式.

备注

此命令等同于这种简写形式: Var /= Value.

除数为零会导致脚本在加载时出现错误信息窗口(如果可能), 否则它会让变量为空.

如果 Var 或 Value 其中一个为空或不以数字开头, 那么在计算中它会被视为 0(不过用在表达式 内部 时例外, 例如 Var := X /= Y).

如果 Var 或 Value 都为小数, 那么最后的结果将为浮点数, 而格式由 SetFormat 决定. 否则, 结果将被截断(例如 19 除以 10 结果为 1).

相关

EnvAdd, EnvSub, EnvMult, SetFormat, 表达式, If Var is [not] Type, SetEnv, 按位运算 (Transform)

示例

设置 MyCount 为自身除以 2.

EnvDiv, MyCount, 2

等同于上面的语句.

MyCount /= 2
