FileGetTime

获取文件或文件夹的时间戳.

FileGetTime, OutputVar , Filename, WhichTime

参数

OutputVar

用来保存获取的 YYYYMMDDHH24MISS 格式的日期时间的输出变量的名称. 这个时间是您的本地时间, 不是 UTC/GMT.

Filename

如果为空或省略, 则使用最内层文件循环的当前文件. 否则, 请指定目标文件或文件夹的名称, 如果未指定绝对路径, 则假定在 %A_WorkingDir% 中.

WhichTime

如果空白或省略, 则默认为 M. 否则, 请指定以下字母之一来设置应检索的时间戳:

  • M = 修改时间
  • C = 创建时间
  • A = 上次访问时间

错误处理

[v1.1.04+]: 此命令失败时会抛出异常. 想了解更多信息, 请参阅运行时错误.

如果遇到问题则 ErrorLevel 被置为 1, 否则为 0.

A_LastError 被设置为操作系统 GetLastError() 函数返回的结果.

备注

请参阅 YYYYMMDDHH24MISS 了解日期和时间的说明.

相关

FileSetTime, FormatTime, If Var is [not] Type, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, 文件循环, EnvAdd (时间计算), EnvSub (时间比较)

示例

获取修改时间, 并将其存储在 OutputVar.

FileGetTime, OutputVar, C:\My Documents\test.doc

获取创建时间, 并将其存储在 OutputVar.

FileGetTime, OutputVar, C:\My Documents\test.doc, C
