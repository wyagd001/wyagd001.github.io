FileEncoding [AHK_L 42+]

为 FileRead, FileReadLine, Loop Read, FileAppend 和 FileOpen 设置默认编码.

FileEncoding , Encoding

编码可以为下列值的其中一个:

  • UTF-8: Unicode UTF-8, 相当于 CP65001.
  • 带小端字节顺序标识的 Unicode UTF-16 等效于 CP1200.
  • UTF-8-RAW 或 UTF-16-RAW: 像上面一样, 但创建新文件时不写入字节顺序标记.
  • CPnnn: 带数值标识符 nnn 的代码页. 请参阅代码页标识符.
  • 空或省略: 系统默认的 ANSI 代码页, 这也是默认设置.

备注

A_FileEncoding 包含了当前的默认编码设置.

如果文件中存在 UTF-8 或 UTF-16 字节顺序标识, 则不会使用默认编码, 除非文件以只写访问打开(即文件的先前内容被丢弃).
