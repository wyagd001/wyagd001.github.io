#KeyHistory

设置在 KeyHistory 窗口中显示的键盘和鼠标事件的最大数目. 您可以将其设置为 0 来禁用按键历史.

#KeyHistory MaxEvents

参数

MaxEvents

在 KeyHistory 窗口中显示的键盘和鼠标事件的最大数目(默认 40, 最大值 500). 指定 0 可以完全禁用按键历史.

备注

因为此设置在脚本开始运行前已经生效了, 所以只需要指定一次(在脚本的任何位置).

因为每次键击或鼠标点击都是由按下事件和弹起事件组成, 所以 KeyHistory 只显示在这里指定的数目一半的 "完整事件". 例如, 如果脚本中含有 #KeyHistory 50, 那么最多显示 25 次的键击和鼠标点击.

相关

KeyHistory, #NoTrayIcon

示例

#1

#KeyHistory 0    ; 禁用按键历史.
#KeyHistory 100  ; 最多保存 100 事件.
