EnvDiv

设置变量的值为它自身除以给定的值.

Deprecated: This command is not recommended for use in new scripts. Use Var := Var / Value or Var /= Value instead.

EnvDiv, Var, Value

参数

Var

要运算的 变量 名.

值

任何整数, 浮点数或 表达式.

备注

此命令等同于这种简写形式: Var /= Value.

除数为零会导致脚本在加载时出现错误信息窗口(如果可能), 否则它会让变量为空.

如果 Var 或 Value 其中一个为空或不以数字开头, 那么在计算中它会被视为 0(不过用在表达式内部时例外, 例如 Var := X /= Y).

如果 Var 或 Value 都为小数, 那么最后的结果将为浮点数, 而格式由 SetFormat 决定. 否则, 结果将被截断(例如 19 除以 10 结果为 1).

相关

EnvAdd, EnvSub, EnvMult, SetFormat, 表达式, If var is [not] type, SetEnv, 按位运算 (Transform)

示例

EnvDiv, MyCount, 2
MyCount /= 2  ; 等同于上面的语句
