如何安装 AutoHotkey

如果您还没有下载 AutoHotkey, 您可以从以下位置之一获得它:

  • https://www.autohotkey.com/ - 当您单击 Download(下载) 时, 会显示多个选项.
  • https://www.autohotkey.com/download/ - 主下载链接是当前的 exe 安装程序, 但也有链接到当前版本的 zip 包和旧版本的存档.

注意: 本教程适用于 AutoHotkey v2.

主下载的文件名为 AutoHotkey_2.0_setup.exe. 运行此文件开始安装 AutoHotkey.

如果您不是计算机的管理员, 则可能需要选择 Current user 选项.

否则, 推荐的选项已经填写, 因此只需单击 Install(安装).

对于 v1 版本的用户: AutoHotkey v2 包含一个启动器, 允许多个版本的 AutoHotkey 共存, 同时共享一个文件扩展名(.ahk). 没有必要将 AutoHotkey v1 和 v2 安装到不同的目录中, 目前也不支持.

如果是为 所有用户 安装, 则需要在出现的标准 UAC 提示中提供管理员同意(换句话说, 点击 Yes).

如果没有什么复杂的情况, AutoHotkey 现在已经安装好了!

一旦安装完成, Dash 会自动显示.

Zip 安装程序

AutoHotkey 也可以从 zip 下载安装.

  1. 使用文件资源管理器打开 zip 文件(不需要解压缩), 或将 zip 文件的内容解压缩到临时目录.
  2. 运行 Install.cmd.
  3. 如果您收到类似 "The publisher could not be verified. Are you sure...?", 单击 Run.
  4. 继续如上所述的安装.

安全提示

您可能会收到一个或多个安全提示, 这取决于几个因素.

Web 浏览器

常见的网络浏览器可能会显示一个警告, 如 "AutoHotkey_2.0_setup.exe 已被阻止, 因为它可能会损害您的设备." 这是一个通用警告, 适用于任何 "commonly downloaded" 的可执行文件类型. 换句话说, 它经常发生在软件的新版本上, 直到更多的用户下载了特定的版本.

为了保持下载, 不同的浏览器使用不同的方法. 在显示下载的地方附近寻找一个菜单按钮, 或者尝试右键单击被阻止的下载.

由于反病毒程序误报, 下载可能会被阻止; 在这种情况下, 请参阅下面的防病毒.

谷歌安全浏览服务(也被其他浏览器使用)已知会显示关于 AutoHotkey 的错误警告. 有关详情, 请参阅 Safe Browsing.

SmartScreen

Microsoft Defender SmartScreen 可能会显示类似 "Windows protected your PC" 的提示. 对于开源开发人员和独立软件供应商(ISV) 的软件来说, 这是很常见的, 特别是在每个新版本发布后不久. 下面这篇由 Louis Kessler 撰写的博客文章很好地描述了这个问题: That’s not very Smart of you, Microsoft

若要继续安装, 请选择 More info 然后选择 Run anyway.
