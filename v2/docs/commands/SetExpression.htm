Var := expression

计算表达式并把结果保存到变量.

Var := expression

参数

Var

保存 表达式 结果的变量的名称.

Expression

有关详情, 请参阅表达式和下面的示例.

注意, 此参数不是可选的. 若要清空该变量, 请指定一个空字符串. 例如: x := ""

备注

单词 true 和 false 是值分别为 1 和 0 的内置变量. 可以使用它们使脚本更易于阅读, 如以下示例所示:

CaseSensitive := false
ContinueSearch := true
