GetMethod

检索方法的实现函数.

Method := GetMethod(Value, Name)

参数

Value

类型: Any

除了 ComObject 之外的任何类型的任何值.

Name

类型: 字符串

要检索的方法的名称.

返回值

类型: 函数对象

函数返回包含方法实现的函数对象.

错误

如果没有找到该方法, 则抛出 MethodError.

备注

方法可以是预先定义的(内置的), 或是在类定义中定义的, 也可以通过调用 DefineMethod 来定义. 搜索中包括继承的方法. 通过 __Call 元-函数实现的方法无法检测, 甚至可能没有相应的函数对象.

调用函数对象时, 必须为通常隐藏的 this 参数提供一个值. 例如, %Method%(Value).

相关

Objects, HasMethod, HasBase, HasProp

示例

检索并报告有关 GetMethod 方法的信息.

method := GetMethod({}, "GetMethod")  ; 这也是一种方法.
MsgBox method.MaxParams  ; 接受 2 个参数, 包括 'this'.
MsgBox method = Func("GetMethod")  ; 在这种情况下, 实际上是同一个对象.
