WinMaximize

将指定的窗口放大到最大尺寸(最大化指定的窗口).

WinMaximize WinTitle, WinText, ExcludeTitle, ExcludeText

参数

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

错误处理

如果找不到窗口, 则抛出异常, 除非使用 group(组) 模式.

备注

使用 WinRestore 还原最大化窗口, 而使用 WinMinimize 来最小化窗口.

如果有特殊类型的窗口无法正确响应, 请尝试使用下面的方法:

PostMessage 0x112, 0xF030,,, WinTitle, WinText  ; 0x112 = WM_SYSCOMMAND, 0xF030 = SC_MAXIMIZE

此函数仅对最上面的匹配窗口起作用, 除非 WinTitle 是 ahk_group GroupName, 在这种情况下, 组中的所有窗口均会受到影响.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.

相关

WinRestore, WinMinimize

示例

#1: 打开记事本, 等到记事本存在后将其最大化.

Run "notepad.exe"
WinWait "Untitled - Notepad"
WinMaximize ; 使用 WinWait 找到的窗口

#2: 按下热键来最大化活动窗口.

^Up::WinMaximize "A"  ; Ctrl+Up
