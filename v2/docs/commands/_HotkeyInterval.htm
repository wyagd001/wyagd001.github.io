#HotkeyInterval

与 #MaxHotkeysPerInterval 一起指定热键激活的速率, 超过此速率会显示警告对话框.

#HotkeyInterval Milliseconds

参数

Milliseconds

类型: 整数

间隔长度, 单位为毫秒.

备注

如果脚本中未使用此指令, 那么它会表现的就像此设置为 2000 那样.

更多细节和备注, 请参阅 #MaxHotkeysPerInterval.

相关

#MaxHotkeysPerInterval

示例

#HotkeyInterval 2000  ; 这是默认值(毫秒).
#MaxHotkeysPerInterval 200
