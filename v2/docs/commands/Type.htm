Type

返回值的类名.

ClassName := Type(Value)

返回值

类型: 字符串

函数返回 Value 的类名. 对于原始值, 返回值是以下之一:

  • String(字符串)
  • Integer(整数)
  • Float(浮点数)

The algorithm for determining a value's class name can be approximated as shown below:

TypeOf(Value)
{
    if (comClass := ComObjType(Value, "Class")) != ""
        return comClass
    try ; `Value is Object` is not checked because it can be false for prototypes.
        if ObjHasOwnProp(Value, "__Class")
            return "Prototype"
    while Value := ObjGetBase(Value)
        if ObjHasOwnProp(Value, "__Class")
            return Value.__Class
    return "Object"
}

对于 COM 包装器对象, the class name can also be determined based on the variant type, as follows:

:

ComObject_Type(obj)
{
    if ComObjType(obj) & 0x2000 ; VT_ARRAY
        return "ComObjArray" ; ComObjArray.Prototype.__Class
    if ComObjType(obj) & 0x4000 ; VT_BYREF
        return "ComValueRef" ; ComValueRef.Prototype.__Class
    if (ComObjType(obj) = 9 || ComObjType(obj) = 13) ; VT_DISPATCH || VT_UNKNOWN
        && ComObjValue(obj) != 0
    {
        if (comClass := ComObjType(obj, "Class")) != ""
            return comClass
        if ComObjType(obj) = 9 ; VT_DISPATCH
            return "ComObject" ; ComObject.Prototype.__Class
    }
    return "ComValue" ; ComValue.Prototype.__Class
}

备注

通常不应该使用此函数来确定一个值是否是数值, 因为数值 字符串 在数学表达式和大多数内置函数中都是有效的. 但是, 在某些情况下, 值的确切类型更为重要. In such cases, consider using Value is Number or similar instead of Type.

要检查一个值是否可以用作数字, 请使用 IsNumber, IsInteger 或 IsFloat 函数.

要检查任何类型的对象(that is, anything which is not a primitive value), 请使用 IsObject 函数.

To check if a value is an instance of a specific class, 请使用 is 运算符. This can be used even with primitive values or to identify COM wrapper objects.

示例

检索并报告存储在 a, b 和 c 中的值的确切类型.

a := 1, b := 2.0, c := "3"
MsgBox Type(a)  ; 整数
MsgBox Type(b)  ; 浮点数
MsgBox Type(c)  ; 字符串
