WinSetEnabled

启用或禁用指定的窗口.

WinSetEnabled Value , WinTitle, WinText, ExcludeTitle, ExcludeText

参数

Value

类型: 字符串或整数

以下值之一(不区分大小写):

  • "On", 1 或 True 打开该设置
  • "Off", 0 或 False 关闭该设置
  • "Toggle" 或 -1 将其设置为与当前状态相反
WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 整数(布尔值)

函数返回 1(true) 表示成功, 或 0(false) 表示失败.

错误处理

如果遇到问题, ErrorLevel 设置为 1, 否则设置为 0. 如果找不到目标窗口或无法应用更改, 则失败.

备注

禁用窗口后, 用户将无法移动它或与其控件进行交互. 另外, 将从 alt-tab 列表中忽略禁用的窗口.

WinGetStyle 页面上的示例 #1 可用于确定是否禁用了窗口.

鼠标光标下的窗口的 ID 可以使用 MouseGetPos 来获取.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.
