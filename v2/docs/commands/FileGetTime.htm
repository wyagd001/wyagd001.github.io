FileGetTime

获取文件或文件夹的时间戳.

Timestamp := FileGetTime(Filename, WhichTime)

参数

Filename

类型: 字符串

目标文件或文件夹的名称, 如果未指定绝对路径则假定在 A_WorkingDir 中. 如果省略, 则使用最内层文件循环的当前文件.

WhichTime

类型: 字符串

要获取的时间戳类别:
M = 修改时间(此参数省略时这是默认的)
C = 创建时间
A = 上次访问时间

返回值

类型: 字符串

函数返回 YYYYMMDDHH24MISS 格式的数字字符串. 这个时间是您的本地时间, 不是 UTC/GMT. 此字符串不应视为数字(不应对其进行数学运算或进行数值比较).

错误处理

失败时抛出异常.

A_LastError 被设置为操作系统 GetLastError() 函数返回的结果.

备注

有关日期和时间的说明, 请参阅 YYYYMMDDHH24MISS.

相关

FileSetTime, FormatTime, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, File-loop, DateAdd, DateDiff

示例

#1

Timestamp := FileGetTime("C:\My Documents\test.doc")  ; 默认获取修改时间.
Timestamp := FileGetTime("C:\My Documents\test.doc", "C")  ; 获取创建时间.
