IsObject

如果指定的值是对象, 则返回非零数字.

IsObject := IsObject(ObjectValue)

参数

ObjectValue

类型: 对象

存储在变量中的, 函数返回的, 存储在另一个对象中的或直接写入的对象.

返回值

类型: 整数(布尔值)

如果指定的值是一个对象, 函数返回 1(true), 如果不是, 返回 0(false).

相关

Objects

示例

object := {key: "value"}

if IsObject(object)
    MsgBox "This is an object."
else
    MsgBox "This is not an object."
