WinGetClass

获取指定窗口的类名.

Class := WinGetClass(WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

WinTitle

类型: 字符串, 整数或对象

窗口标题或标识目标窗口的其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果使用此参数, 那么它必须是目标窗口中单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 设置为 ON 时, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

排除标题中含有此参数值的窗口.

ExcludeText

类型: 字符串

排除文本中含有此参数值的窗口.

返回值

类型: 字符串

函数返回指定残窗口的类名. 如果没有匹配的窗口, 则返回一个空字符串.

备注

仅获取类名(返回值中不包含前缀 "ahk_class").

窗口标题和文本是区分大小写的. 只有在打开 DetectHiddenWindows 设置时, 才会检测隐藏窗口.

相关

WinGetTitle, Win 函数

示例

#1

MsgBox "The active window's class is " WinGetClass("A")
