ControlGetHwnd

返回指定控件的唯一 ID.

Hwnd := ControlGetHwnd(Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

Control

类型: 字符串, 整数或对象

控件的 ClassNN, 文本或 HWND, 或具有 Hwnd 属性的对象. 有关详情, 请参阅 Control 参数.

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 整数

此函数返回指定控件的窗口句柄(HWND).

错误处理

如果找不到窗口或控件, 则抛出异常.

备注

控件的 HWND 通常与 PostMessage, SendMessage 和 DllCall 一起使用. 另外, 还可以通过 MouseGetPos 来获取控件的 HWND. 最后, 控件的 HWND 可以直接在 WinTitle 参数中使用. 这也适用于隐藏控件, 即使 DetectHiddenWindows 为 Off.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.

相关

WinGetID, Hwnd 属性(GuiControl 对象), Control 函数

示例

示例 #1

#1: 检索记事本的 Edit 控件的唯一 ID:

editHwnd := ControlGetHwnd("Edit1", "ahk_class Notepad")
