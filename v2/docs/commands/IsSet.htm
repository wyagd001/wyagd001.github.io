IsSet

如果指定变量已经被赋值, 则返回一个非零数字.

IsSet := IsSet(Var)

参数

Var

类型: 变量

变量的引用. 例如: IsSet(MyVar).

返回值

类型: 整数(布尔值)

如果 Var 已经被赋值, 函数返回 1(true), 否则返回 0(false).

备注

未赋值的变量也称为未初始化变量.

默认情况下, 未初始化的变量包含一个空字符串, 一些脚本作者可能会选择依赖于此. 但是, 如果使用了 #Warn UseUnset, 除了 IsSet 以外的绝大多数其他方法访问未初始化的变量都有可能触发警告消息. IsSet 可用于确定一个变量是否需要初始化, 例如在第一次使用时初始化一个全局变量或静态变量.

已知限制: 如果 MsgBox 警告模式生效, 触发给定变量的警告消息的副作用是初始化该变量以防止进一步的警告. 这将导致 IsSet 为后续调用返回 true.

相关
