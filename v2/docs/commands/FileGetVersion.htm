FileGetVersion

检索文件的版本.

Version := FileGetVersion(Filename)

参数

Filename

类型: 字符串

目标文件名. 如果未指定完整路径, 则此函数使用系统 LoadLibrary 函数所指定的搜索顺序. 如果省略, 则将使用括在 File-Loop 最内层的当前文件.

返回值

类型: 字符串

函数返回指定文件的版本号.

错误处理

失败时抛出异常.

A_LastError 设置为操作系统 GetLastError() 函数的结果.

备注

大多数非可执行文件(甚至一些 exe 文件) 都没有版本, 因此在这些情况下返回值将为空.

相关

FileGetAttrib, FileSetAttrib, FileGetTime, FileSetTime, FileGetSize, File-loop

示例

#1: 读取文件的版本, 并将其存储在 Version.

Version := FileGetVersion("C:\My Application.exe")

#2: 检索位于 AutoHotkey 安装目录下的文件 "AutoHotkey.exe" 的版本, 并将其存储在 Version.

Version := FileGetVersion(A_ProgramFiles "\AutoHotkey\AutoHotkey.exe")
