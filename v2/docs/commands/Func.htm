Func

检索指定函数的引用.

FunctionReference := Func(FunctionName)

参数

FunctionName

类型: 字符串

要检索引用的函数名称. FunctionName 必须明确的存在于脚本中, 或是内置的.

返回值

类型: 对象

函数返回 FunctionName 的引用; 也就是, Func 对象或闭包对象. 如果 FunctionName 不是内置的, 并且在脚本中没有显式存在(通过诸如 #Include 或对库函数的非动态调用), 它将返回一个空字符串.

如果 FunctionName 是带有自由变量的套嵌函数, 则创建并返回一个新的闭包. 它的性质和 Func 是一样的.

备注

返回的对象可用于调用函数或检索信息(如 参数的最小和最大数目).

相关

函数引用, Func 对象, 闭包, IsFunc

示例

#1

; 检索函数名为 "StrLen" 的引用.
fn := Func("StrLen")

; 显示函数的相关信息.
MsgBox fn.Name "() is " (fn.IsBuiltIn ? "built-in." : "user-defined.")
