DirDelete

删除文件夹.

DirDelete DirName , Recurse

参数

DirName

类型: 字符串

要删除的目录名, 如果未指定绝对路径, 则假定在 A_WorkingDir 中.

Recurse

类型: 布尔值

如果省略, 它默认为 0(false).

如果为 false, 不 移除 DirName 中包含的文件和子目录. 此时如果 DirName 不是空的, 则不进行操作并且抛出异常.

如果为 true, 移除所有文件和子目录(类似于 Windows 命令 "rmdir /S").

错误处理

如果发生错误, 则抛出异常.

相关

DirCreate, FileDelete

示例

删除目录, 但仅限于空目录.

DirDelete "C:\Download Temp"

删除目录, 包括其中的文件和子目录.

DirDelete "C:\Download Temp", 1
