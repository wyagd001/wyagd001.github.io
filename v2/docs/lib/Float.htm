Float

将数字字符串或整数值转换为浮点数.

FltValue := Float(Value)

返回值

类型: 浮点数

函数返回将 Value 转换为纯浮点数(类型名称为 "Float") 的结果, 或 Value 本身(如果它已经是正确的类型).

备注

如果值无法转换, 则抛出 TypeError.

要确定一个值是否可以转换为浮点数, 使用 IsNumber 函数.

Float 实际上是一个类, 但可以作为一个函数被调用. Value is Float 可以用来检查一个值是否是一个纯浮点数.
