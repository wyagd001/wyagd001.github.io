ListHotkeys

显示当前脚本使用的热键, 不论它们的子程序当前是否运行, 也不论它们是否使用键盘或鼠标钩子.

ListHotkeys

此函数等同于在主窗口中选择 View->Hotkeys 菜单项.

如果某个热键已被 Hotkey 函数禁用, 则在列表中它会显示为 OFF 或 PART("PART" 意思是此热键有某些变体被禁用了).

如果热键的任何变体使用了非 0 级别的 #InputLevel, 将显示此级别(或最小及最大级别).

如果任何热键的子程序当前正在运行, 则显示该热键的线程总数.

最后, 还会显示热键的类型, 为下列的其中一个:

  • reg: 此热键是通过操作系统的 RegisterHotkey() 函数来实现的.
  • reg(no): 与上面相同, 不过热键当前不可用(由于不支持, 被禁用或挂起).
  • k-hook: 此热键是通过键盘钩子来实现的.
  • m-hook: 此热键是通过鼠标钩子来实现的.
  • 2-hooks: 此热键需要以上两种钩子.
  • joypoll: 此热键是通过定期轮询控制器来实现的.
