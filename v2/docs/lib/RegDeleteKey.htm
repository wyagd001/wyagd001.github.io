RegDeleteKey

从注册表中删除子键.

RegDeleteKey KeyName

参数

KeyName

类型: 字符串

注册表键的全名, 例如 "HKLM\Software\SomeApplication".

必须以 HKEY_LOCAL_MACHINE(或 HKLM), HKEY_USERS(或 HKLM), HKEY_CURRENT_USER(或 HKLM), HKEY_CLASSES_ROOT(或 HKLM) 或 HKEY_CURRENT_CONFIG(或 HKLM) 开始.

要访问远程注册表, 请在前面加上计算机名和反斜杠, 例如 "\\workstation01\HKLM".

只有在注册表循环时, 才能省略 KeyName, 在这种情况下, 默认为当前循环项目的键. 如果项目是子键, 则默认使用该子键的全名.

错误处理

失败时抛出 OSError.

A_LastError 被设置为操作系统 GetLastError() 函数返回的结果.

备注

警告: 对注册表进行删除操作有潜在的风险 - 操作前请三思!

要检索和操作多个注册表键或值, 请考虑使用注册表循环.

在注册表循环中, RegDeleteKey 没必要删除当前循环的项目. 如果项目是子键, 则 RegDeleteKey() 将删除键本身. 如果项目是值, 则 RegDeleteKey() 删除 包含 该值的键, 包括所有子键和值.

有关如何访问远程计算机注册表的详情, 请参阅注册表循环中的备注.

从 32 位脚本中删除注册表的 64 位部分中的条目, 反之亦然, 请使用 SetRegView.

相关

RegCreateKey, RegCreateKey, RegDelete, RegRead, RegWrite, Registry-loop, SetRegView, IniDelete

示例

从注册表中删除一个值.

RegDeleteKey "HKEY_LOCAL_MACHINE\Software\SomeApplication"
