DriveGetFileSystem

返回指定驱动器的文件系统的类型.

FileSystem := DriveGetFileSystem(Drive)

参数

Drive

类型: 字符串

驱动器字母后跟着冒号和可选的反斜杠, 或 UNC 名称, 如 "\server1\share1".

返回值

类型: 字符串

函数返回 Drive 的文件系统的类型. 可能的由系统定义的值; 它们包括(但不限于) 以下内容: NTFS, FAT32, FAT, CDFS(通常表示 CD), 或 UDF(通常表示 DVD).

错误处理

失败时例如驱动器不包含格式化的媒体, 则抛出异常.
