ExitApp

终止脚本.

ExitApp ExitCode

参数

ExitCode

类型: 整数

如果省略, 则默认为 0(零一般用来表示成功). 否则, 请指定一个介于 -2147483648 和 2147483647 之间的整数, 在脚本退出时返回给它的调用者. 此代码可以被任何调用脚本的程序访问, 例如另一个脚本(使用 RunWait) 或批处理(.bat) 文件.

备注

这等同于从脚本的托盘菜单或主菜单中选择了 "Exit".

任何由 OnExit 注册的函数在脚本终止之前都会被调用. 如果这样的函数返回一个非零整数, 脚本不会终止; 作为代替, 当前线程退出就像调用了 Exit 一样.

终止脚本并不等同于退出每个线程. 例如, Finally 块不会被执行, 而对于局部变量所包含的对象 __Delete 不会被调用.

在不是持续运行的脚本中, ExitApp 通常是不必要的.

相关

Exit, OnExit, Persistent

示例

按下热键来终止当前脚本.

#x::ExitApp  ; Win+X
