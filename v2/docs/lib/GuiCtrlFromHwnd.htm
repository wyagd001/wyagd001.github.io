GuiCtrlFromHwnd

检索与指定的窗口句柄关联的 GUI 控件的 GuiControl 对象.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

参数

Hwnd

类型: 整数

GUI 控件的窗口句柄(HWND), 或控件的子窗口(例如 ComboBox 的 Edit 控件) 的窗口句柄. 该控件必须是由当前脚本通过 MyGui.Add 创建的.

返回值

类型: Object 或字符串(空)

函数返回与指定的 HWND 相关联的 GuiControl 对象, 如果没有该对象或 HWND 无效, 则返回一个空字符串.

备注

例如, 能通过 GuiCtrl.Hwnd, MouseGetPos 或 OnMessage 来检索 GUI 控件的 HWND.

相关

Gui(), Gui 对象, GuiControl 对象, GuiFromHwnd, 控件类型, ListView, TreeView, Menu 对象, 控件函数, MsgBox, FileSelect, DirSelect

示例

请参阅 Gui 对象页面上的 ToolTip 示例.
