GetKeyName

检索按键的名称或文本.

Name := GetKeyName(KeyName)

参数

KeyName

类型: 字符串

这可以是键盘上的任何单个字符, 也可以是按键列表中的一个键名. 例如: B, 5, LWin, RControl, Alt, Enter, Escape.

或者, 可以指定显式虚拟键码(如 vkFF), 或显式扫描码(如 sc01D), 或 VK 和 SC 的组合(按该顺序, 如 vk1Bsc001). 请注意, 这些代码必须是十六进制的.

返回值

类型: 字符串

函数返回指定按键的名称, 如果按键无效或没有名称, 则返回空白.

相关

GetKeyVK, GetKeySC, GetKeyState, Key List

示例

检索并报告 Esc 的英文名称.

MsgBox GetKeyName("Esc") ; 显示 Escape
MsgBox GetKeyName("vk1B") ; 也显示 Escape
