#Hotstring

改变热字串的选项或终止符.

#Hotstring NoMouse
#Hotstring EndChars NewChars
#Hotstring NewOptions

参数

NoMouse

类型: 字符串

阻止鼠标点击重置热字串识别器, 如这里描述. 副作用是, 这样也阻止了热字串需要的鼠标钩子(尽管如果脚本因其他目的(如鼠标热键) 需要鼠标钩子, 它仍然会被安装). 在脚本的任意位置使用 #Hotstring NoMouse 会影响所有的热字串, 而不只是影响在它后面的那些.

EndChars NewChars

类型: 字符串

指定单词 EndChars 后跟着单个空格和新的终止符. 例如:

#Hotstring EndChars -()[]{}':;"/\,.?!`n`s`t

由于新的结束字符在整个脚本中都是全局有效的 -- 不管 EndChars 指令出现在哪里 -- 所以不需要多次指定 EndChars.

结束符的最大数目是 100. 超出此长度的字符被忽略.

要使制表符或空格符成为终止符, 请在列表中包含 `t 或 `s.

NewOptions

类型: 字符串

指定新选项, 如 Hotstring 选项中描述的那样. 例如: #Hotstring r s k0 c0.

与上面的 EndChars 不同, 当以这种方式使用时, #Hotstring 指令是与位置有关的. 换句话说, 脚本中的多个热字串可以具有不同的默认选项, 如下例所示:

::btw::by the way

#Hotstring r c  ; 后面的所有热字串会使用 "send raw" 且默认区分大小写.
::al::airline
::CEO::Chief Executive Officer

#Hotstring c0  ; 让从这里往后的所有热字串不区分大小写.

备注

和其他指令一样, #Hotstring 不能有条件地执行.
