#NoTrayIcon

禁止显示托盘图标.

#NoTrayIcon

在脚本的任何地方指定此参数将防止在启动脚本时显示该脚本的托盘图标(也适用于编译成 EXE 的脚本).

如果您在含热键的脚本中使用此指令, 那么可能需要为 ExitApp 函数绑定一个热键. 否则, 将没有简单的办法退出程序(在不重启计算机或强制关闭进程的情况下). 例如: #x::ExitApp.

托盘图标可以在脚本执行期间的任何时候消失或重新出现, 方法是为 A_IconHidden 赋值 true 或 false. 在脚本顶部使用 A_IconHidden := true 的唯一缺点是, 在脚本首次启动时, 托盘图标可能会短暂可见. 为了避免这种情况, 请使用 #NoTrayIcon.

当托盘图标隐藏时, 内置变量 A_IconHidden 的值为 1, 否则为 0, 并且可以为其赋值以显示或隐藏图标.

和其他指令一样, #NoTrayIcon 不能有条件地执行.

相关

Tray Icon, TraySetIcon, A_IconHidden, A_IconTip, ExitApp

示例

使脚本启动时没有托盘图标.

#NoTrayIcon
