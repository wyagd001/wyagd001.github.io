Any

理论上, AutoHotkey 的类型层次结构的根源在于 "Any" 类型. 所有其他类型都是 Any 的子类, 除了 ComObject(在本页中不作进一步考虑). 目前 Any 类型仅作为原型对象存在, 它位于每个值的基对象链的末端. 有关详细信息, 请参阅原始值.

Any 原型对象定义了适用于所有值和对象的方法和属性, 被覆盖改写的除外. 原型对象本身本质上是一个 Object, 但是没有 base, 因此不能识别为对象的实例.

方法:

  • GetMethod
  • HasBase
  • HasMethod
  • HasProp

属性:

  • Base

函数:

  • ObjGetBase

方法

GetMethod

检索方法的实现函数.

Value.GetMethod(Name)

除非被覆盖, 否则这个方法完全相同于 GetMethod(Value, Name).

HasBase

如果 BaseObj 在 Value 的基对象链中, 则返回 true, 否则返回 false.

Value.HasBase(BaseObj)

除非被覆盖, 否则这个方法完全相同于 HasBase(Value, BaseObj).

HasMethod

如果该值具有使用此名称的方法, 则返回 true, 否则返回 false.

Value.HasMethod(Name)

除非被覆盖, 否则这个方法完全相同于 HasMethod(Value, Name).

HasProp

如果值具有使用此名称的属性, 则返回 true, 否则返回 false.

Value.HasProp(Name)

除非被覆盖, 否则这个方法完全相同于 HasProp(Value, Name).

属性

Base

检索值的基对象.

BaseObj := Value.Base

对于原始值, 返回值是与 Type(Value) 对应的预定义原型对象.

另请参阅: ObjGetBase, ObjSetBase, Obj.Base

函数

ObjGetBase

返回值的基对象.

BaseObj := ObjGetBase(Value)

不调用元函数或属性函数. 覆盖 Base 属性不会影响此函数的行为.

如果没有基, 则返回值为空字符串. 只有 COM 对象和 Any 原型本身没有基.

另请参阅: Base, ObjSetBase, Obj.Base
