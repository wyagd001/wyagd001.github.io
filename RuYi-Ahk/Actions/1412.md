# [](#header-2) 如意托盘菜单

**动作编号**: 1412  
**动作名称**: 如意托盘菜单  
**动作作用的对象**: 无条件  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 显示如意的托盘菜单(右击托盘图标显示的菜单)  
**动作截图**: 无   
**动作内容**: Cando|TM_show  
执行如意的内置的标签 "TM_show", 为如意内置动作
