# [](#header-2) Chrome 设置

**动作编号**: 1145  
**动作名称**: Chrome 设置  
**动作作用的对象**: Chrome  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 打开 Chrome 浏览器的设置页面  
**动作截图**: 无  
**动作内容**: run|"%B_Autohotkey%" "%A_ScriptDir%\外部脚本\窗口处理\Chrome地址栏打开.ahk" "chrome://settings/"  
以指定网址作为参数, 执行外部脚本文件 "Chrome地址栏打开.ahk", ATA 内置动作  

**代码及详细解释**:  
利用 Chrome 浏览器的地址栏打开指定的网址, 打开浏览器设置页面.
