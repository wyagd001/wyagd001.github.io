# [](#header-2) Base64加密

**动作编号**: 1337  
**动作名称**: Base64加  
**动作作用的对象**: 选中文本  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 将选中的文本进行Base64加密  
**动作截图**:  
  ![Base64加](img1/1337.png)  
**动作内容**: run|"%B_Autohotkey%" "%A_ScriptDir%\外部脚本\文本处理\Base64加密.ahk" "%CandySel%"  
以选中文本作为参数, 执行外部脚本文件 "Base64加密.ahk", ATA 内置该动作  

**代码或详细解释**:  
选中文本, 执行动作, 弹出一个显示选中文本加密得到的 Base64 值的窗口.
