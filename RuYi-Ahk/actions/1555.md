# [](#header-2) 农历时间

**动作编号**: 1555  
**动作名称**: 农历时间  
**动作作用的对象**: 无条件  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 显示农历时间  
**动作截图**:  
  ![农历时间](img1/1555.png)  
**动作内容**: run|"%B_Autohotkey%" "%A_ScriptDir%\外部脚本\工具类\时间_显示农历.ahk"  
无参数, 执行外部脚本文件 "时间_显示农历.ahk", ATA 内置该动作    

**代码或详细解释**:  
若额外任务栏存在则在其顶部显示, 否则显示一个对话框.
